@par Application Description

This is a TouchGFX Animation Example for the STM32HB3I-DK board, created by ST.The Animation Example shows how to display,
start and stop an animation via a range of images. The example shows how to configure the animation to a particular frame
rate and how to reverse the order of the animation. The AnimatedImage starts or stops when pressing the button. 
If the AnimatedImage has been in motion, the AnimatedImage starts form where it was stopped.Each time the animation ends,
an interaction is implemented to restart and reverse the animation.

Requirements:
	Resolution: 480 * 272
	BPP: 24

@par How to use it ?

Please use STM32CubeProgrammer utility to flash the application's HEX file to the STM32 target.
